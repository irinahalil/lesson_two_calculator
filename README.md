# Семейный бюджет (статический сайт)

Проект состоит из двух страниц:

- `index.html` — небольшой приветственный лендинг с CTA к функционалу
- `Калькулятор.html` — калькулятор семейного бюджета (доходы/расходы, сводка, LocalStorage и переключение темы)

## Как запустить

Так как сайт статический, достаточно открыть файлы в браузере:

1. Откройте `index.html`
2. Нажмите `Открыть калькулятор`

Либо напрямую откройте `Калькулятор.html`.

## Тема (светлая/тёмная)

Переключение сделано кнопкой в правом нижнем углу.
Выбранная тема сохраняется в `LocalStorage` под ключом `familyBudgetTheme`.
